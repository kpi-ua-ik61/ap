# Lab 03

## On UNIX systems

Compile with

```sh
$ make
```

Run with

```sh
$ ./lab_03
```

Preferable to compile and run under UNIX systems

## On other OS

To compile under other OS do:

```sh
$ ... lab_03.c error.c func.c zadanie1.c zadanie2.c zadanie3.c
````

Also include header file:

```sh
$ ... -I header.h
```
