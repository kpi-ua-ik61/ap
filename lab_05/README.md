# Lab 05

## On UNIX systems

Compile with

```sh
$ make
```

Run with

```sh
$ ./lab_05
```

Preferable to compile and run under UNIX systems

## On other OS

To compile under other OS do:

```sh
$ ... lab_05.c error.c ft_atoi.c ft_strlen.c ft_strcmp.c
```

Also include header file:

```sh
$ ... -I header.h
```
