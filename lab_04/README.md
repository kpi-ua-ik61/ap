# Lab 04

## On UNIX systems

Compile with

```sh
$ make
```

Run with

```sh
$ ./lab_04
```

Preferable to compile and run under UNIX systems

## On other OS

To compile under other OS do:

```sh
$ ... lab_04.c error.c func.c zadanie1.c zadanie2.c
```

Also include header file:

```sh
$ ... -I header.h
```
